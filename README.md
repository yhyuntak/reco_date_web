# Reco_date_web

This project was bootstrapped with Vite, React, TypeScript, Yarn, and ESLint.

## Getting Started

### Development
```
yarn dev
```

### Build
```
yarn build
```

### Lint
```
yarn lint
```

## Features
- React 19
- TypeScript
- Vite for fast development
- ESLint for code linting
- Yarn as the package manager

---

For more details, see the documentation for each tool.
